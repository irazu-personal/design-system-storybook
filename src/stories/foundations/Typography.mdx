# Typography

Type tokens from the Figma **Typography** collection. Each Figma text style maps to a semantic token
(`--semantic-typography-text-style-*`) composed from `font-size`, `line-height`, and `font-weight`
primitives; platform UI uses **Figtree**, code uses **Source Code Pro**.
